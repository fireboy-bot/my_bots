# 🏰 Числяндия — Образовательный Telegram-бот

**Версия:** 1.0.0  
**Дата последнего обновления:** 21 февраля 2026  
**Статус:** ✅ Готов к продакшену

## 📋 Описание

Telegram-бот для обучения детей математике через игру.  
Путешествие по островам Числяндии, битвы с боссами, система достижений.

## 🎯 Основные возможности

- ✅ **SQLite база данных** — надёжное хранение прогресса
- ✅ **Кэш аватарок** — мгновенная загрузка после первой
- ✅ **Атмосферное интро** — легенда мира с героями
- ✅ **4 острова** — Сложение, Вычитание, Умножение, Деление
- ✅ **5 основных боссов** + Истинный Владыка (3 фазы)
- ✅ **Система уровней и XP** — прогрессия игрока
- ✅ **Бэкап/Restore базы** — защита данных
- ✅ **Полное логирование** — с ротацией файлов
- ✅ **Graceful shutdown** — корректное завершение

## 🚀 Быстрый старт

```bash
# 1. Установи зависимости
pip install python-telegram-bot python-dotenv

# 2. Создай .env файл
cp .env.example .env
# Отредактируй: добавь BOT_TOKEN и ADMIN_IDS

# 3. Запусти бота
py manyunya_bot.py
