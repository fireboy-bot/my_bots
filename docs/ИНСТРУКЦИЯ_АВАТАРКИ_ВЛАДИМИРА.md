## 🔧 troubleshooting

### ❌ Проблема: "Файл не найден"

**Лог:**
```
⚠️ Файл не найден: C:\...\images/vladimir_approve.jpg
```

**Решение:**
```powershell
# Проверь что файл существует:
dir images\vladimir_approve.jpg

# Если нет — скопируй файл в папку images/
```

---

### ❌ Проблема: "file_id=None"

**Лог:**
```
🔍 Cache lookup: key=vladimir_approve, file_id=❌
```

**Решение:**
```powershell
# Перезагрузи кэш:
Remove-Item data\avatar_cache.json -Force

# Перезапусти бота:
py manyunya_bot.py

# Смотри логи — должно загрузить файл
```

---

### ❌ Проблема: Аватарка не меняется

**Проверь:**
1. Файл лежит в `images/`?
2. Кэш обновлён (удалён `avatar_cache.json`)?
3. `VLADIMIR_MOODS` обновлён в коде?
4. Бот перезапущен?
